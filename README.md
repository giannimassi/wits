# wits

Composable reasoning skills for AI agents. Structured thinking primitives, facilitated multi-agent discussion, and expert persona recruitment.

## Skills

- **think** — 12 reasoning primitives (assumption-audit, pre-mortem, steelman, etc.)
- **discuss** — facilitated multi-agent discussion with structured roles
- **recruit** — expert persona recruitment and caching

## Install

```bash
claude plugin install giannimassi/wits
```

## Usage

Skills are invoked as `/wits:think`, `/wits:discuss`, `/wits:recruit`.
